📌 Verze a změny aplikace

Přehled úprav, aby bylo jasné, co se v aplikaci měnilo.

Historie verzí
29.06.2026 – v4.6

Přidána samostatná galerie s alby, nahráváním fotek jako náhledů a mazáním fotek i celých alb. Všechny rozbalovací části v záložkách se po otevření záložky automaticky zavírají, aby se nemuselo zbytečně sjíždět a zavírat ručně.

29.06.2026 – v4.5

Přidáno finanční porovnání výdělků proti nákupům, rozdíl hotově i včetně hodnoty skladu, historie stáčení a prodeje v kusech, kilogramech a korunách.

29.06.2026 – v4.4

Odstraněn dlouhý popis pod názvem aplikace. Přidána samostatná záložka Verze s historií úprav.

29.06.2026 – v4.3

Opravena vlastní spotřeba medu: jde upravit i smazat, odečítá se ze skladu a ve skladu je vidět vyrobeno / prodáno / darováno / vlastní / zbývá k prodeji.

29.06.2026 – v4.2

Opraveny úpravy prodeje, darování a vlastní spotřeby u šarží, které už mají 0 kusů skladem.

29.06.2026 – v4.1

Opraveno zobrazování smazaných zákazníků v historii, rychlá kontrola už neukazuje falešně slabé včelstvo a graf stanovišť počítá i ručně přiřazené úly.

Předchozí verze

Zákazníci, objednávky, prodej medu, darování, vlastní spotřeba, počasí přes GPS / stanoviště, fotky kontrol, zálohy a import starších JSON dat.

Co kontrolovat po nahrání na GitHub
Po nahrání nové verze ověř:

1) Přidání vlastní spotřeby odečte kusy ze skladu. 2) Úprava vlastní spotřeby funguje i u poslední sklenice. 3) Záloha JSON jde exportovat a znovu importovat. 4) Počasí funguje na GitHub Pages přes HTTPS.
